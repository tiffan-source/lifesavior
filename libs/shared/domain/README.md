# @shared/domain

## 📍 Rôle
Librairie socle partagée contenant les abstractions de base du système (erreurs, types primitifs).

## 📦 Contenu Clé
- **`CoreError`** : Classe abstraite de base pour toutes les erreurs métier typées (code, message, metadata).

## Building

Run `nx build shared-domain` to build the library.

## Running unit tests

Run `nx test shared-domain` to execute the unit tests via [Vitest](https://vitest.dev/).
